### 1. Создайте интеграцию в amoCRM, для этого:
  Зайдите в amoМаркет -> сверху справа нажмите ... -> создать интеграцию -> внешняя интеграция. Далее в поле ссылка для перенаправления введите адресс вашего сайта (если нет, то любой, например https://localhost.com). 
  Название интеграции и описание какое хотите.
  Проверьте, чтобы интеграция была включена
### 2. Зайдите в интеграцию -> ключи и доступы. Сгенерируйте и сохраните секретный ключ. Получите ID интеграции и код авторизации.
### 3. В файл .env введите данные. 
  Ваш субдомен (только имя, без https:// и без .amocrm.ru)
  SUBDOMAIN=your_subdomain
  
  ID интеграции
  CLIENT_ID=your_client_id
  
  Cекретный ключ
  CLIENT_SECRET=your_client_secret
  
  Код авторизации
  AUTH_CODE=your_auth_code
  
  Redirect URI — должен совпадать с тем, что указали в интеграции
  REDIRECT_URI=https://localhost.com
  
  ACCESS TOKEN и REFRESH TOKEN появятся автоматически после первого запуска (пока оставьте пустыми)
  ACCESS_TOKEN=
  REFRESH_TOKEN=

### 4. Запустите файл amocrm_auth.py. После этого в файле .env должны появиться данные в ACCESS_TOKEN и REFRESH_TOKEN
### 5. Готово. Теперь можете создавать и изменять сделки в файле main.py или с помощью Swagger UI (для этого введите команду ``` uvicorn main:app --reload ```  и зайдите на http://127.0.0.1:8000/docs)
